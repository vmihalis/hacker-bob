# Hacker Bob v1.1.1 Release Notes

## Highlights

`v1.1.1` was a Claude slash-menu hygiene patch. The release line had parallel Bob entry points because skill-backed workflows and command shims advertised similar actions under different names.

## What Changed

- Removed redundant Claude hunt/status/debug command shims from the release line.
- Kept only the update command shim, because update is installer-backed rather than skill-backed.
- Added upgrade cleanup for legacy `commands/bob/{hunt,status,debug}.md` files.
- Added uninstall coverage for both current and legacy Claude command layouts.

## Superseded

`v1.1.1` tried colon-form Claude skill names (`bob:hunt`, `bob:status`, `bob:debug`). Claude Code rejects colons in skill `name:` frontmatter, so `v1.1.2` supersedes this release with hyphen-form names.
