# Hacker Bob v1.1.2 Release Notes

## Highlights

`v1.1.2` finishes the host command cleanup with one visible entry per Bob workflow. Claude skills now use hyphen names because Claude Code accepts lowercase letters, numbers, and hyphens in skill `name:` frontmatter. Codex now uses matching direct skill names instead of plugin-namespaced `$hacker-bob:*` skills.

## Claude User-Facing Changes

- `/bob-hunt target.com` instead of `/bob:hunt target.com`
- `/bob-status` instead of `/bob:status`
- `/bob-debug` instead of `/bob:debug`
- `/bob-update` instead of `/bob:update`

The MCP server name (`bountyagent`) and every `mcp__bountyagent__*` tool name is unchanged.

## Codex User-Facing Changes

- `$bob-hunt target.com` instead of `$hacker-bob:hunt target.com`
- `$bob-status` instead of `$hacker-bob:status`
- `$bob-debug` instead of `$hacker-bob:debug`
- `$bob-update` instead of `$hacker-bob:update`

The Codex adapter installs those skills into `~/.codex/skills`. The local `.codex/plugins/hacker-bob` plugin remains in place for MCP wiring and command wrappers, but it no longer declares plugin-scoped skills.

## File Layout

- `.claude/skills/bountyagent/` -> `.claude/skills/bob-hunt/`
- `.claude/skills/bountyagentstatus/` -> `.claude/skills/bob-status/`
- `.claude/skills/bountyagentdebug/` -> `.claude/skills/bob-debug/`
- `.claude/commands/bob/update.md` -> `.claude/commands/bob-update.md`
- `adapters/codex/hacker-bob/skills/{hunt,status,debug,update}/` -> `adapters/codex/skills/bob-{hunt,status,debug,update}/`
- `.codex/plugins/hacker-bob/skills/*` -> direct `~/.codex/skills/bob-*` installs

## Upgrade Safety

The installer removes legacy Claude and Codex paths on upgrade:

- `.claude/commands/bob/hunt.md`
- `.claude/commands/bob/status.md`
- `.claude/commands/bob/debug.md`
- `.claude/commands/bob/update.md`
- `.claude/skills/bountyagent`
- `.claude/skills/bountyagentstatus`
- `.claude/skills/bountyagentdebug`
- `.codex/plugins/hacker-bob/skills/{hunt,status,debug,update}`
- `.codex/plugins/hacker-bob/skills/hacker-bob-{hunt,status,debug,update}`
- `.codex/plugins/hacker-bob/commands/{hunt,status,debug,update}.md`
- `~/.codex/skills/hacker-bob-{hunt,status,debug,update}`

After installing or updating, fully restart Claude Code or Codex in the target project so the host reloads skills, commands, MCP config, hooks, and statusline setup.

## Version Semantics

The canonical `hacker-bob` package, `hacker-bob-cc` compatibility package, Codex plugin manifest, and neutral `.hacker-bob/VERSION` metadata all report `1.1.2`.
