# Roadmap

This roadmap is intentionally short and public. It describes the next areas where community contributions are most useful, without committing to exact release dates.

## Install Reliability

Improve installer diagnostics, update recovery, uninstall confidence, and platform-specific guidance for macOS, Linux, Windows, and common host-adapter project layouts.

## Docs And Media

Add real sanitized screenshots and short demos for first install, `hacker-bob doctor`, `/bob-status`, and a safe `/bob-hunt` start against a controlled target.

## MCP Diagnostics

Make MCP startup, tool registration, Claude config wiring, and statusline failures easier to diagnose from `hacker-bob doctor` and `/bob-debug`.

## Hunter Quality

Improve attack-surface ranking, duplicate suppression, verification prompts, and report evidence quality so fewer low-confidence findings reach the final report.

## Security Hardening

Continue tightening path safety, session storage boundaries, SSRF controls, secret redaction, npm release checks, and dependency hygiene.

## Contributor Onboarding

Document focused development tasks, test fixtures, generated prompt surfaces, review expectations, and safe local lab workflows for first-time contributors.
